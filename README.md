# Kids Learning Adventure Full Package

This combines the typing game and coding expansion in one folder.

## Structure

- `index.html` — main launcher
- `typing/index.html` — typing games
- `coding/index.html` — coding-thinking games
- `js/sound.js` — shared reusable sound module
- `sounds/*.mp3` — ready-to-use sound effects

## Installation

Back up the current folder, then upload all files and folders from this package without changing the structure.

Open `index.html`.

Browsers normally require one click before audio can play.
